# Fund Transfer System

A full-stack fund transfer application built with:

- React + Tailwind (Frontend)
- Node.js + Express (Backend)
- PostgreSQL (Database)
- Immutable audit logging
- JWT-based authentication

## Features
- User login
- Fund transfer between users
- Atomic transactions
- Immutable audit logs
- Transaction history

## Tech Stack
- React (Vite)
- Node.js
- Express
- PostgreSQL
- Tailwind CSS

## Deployment
- Frontend: Vercel
- Backend: Render
- Database: PostgreSQL (Render)
